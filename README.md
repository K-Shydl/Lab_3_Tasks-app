## Варіант 4: "Мої завдання"

### Екрани

**1. TasksListScreen** - Список завдань
- Список всіх завдань (назва + пріоритет)
- Кнопка "+" для додавання нового завдання
- При клику на завдання → перехід на DetailsScreen з ID завдання

**2. AddTaskScreen (Додавання завдання)**

**Призначення:** Форма для створення нового завдання.

**Функціональність:**
- Поле для введення назви завдання
- Поле для опису
- Список вибору пріоритету (Низький/Середній/Високий)
- Кнопка "Створити" - додає завдання і повертається на Екран 1
- Кнопка "Скасувати" - повертається назад без збереження

**Переходи:**
- `AddTaskScreen` → `TasksListScreen` (кнопка "Створити" або "Скасувати")

**3. DetailsTaskScreen** - Деталі завдання
- Отримує ID завдання як аргумент
- Показує назву, опис, пріоритет та статус
- Чекбокс для позначення завдання як виконаного
- Кнопка "Редагувати" (переходить на AddScreen з ID завдання)
- Кнопка "Видалити" (видаляє завдання і повертається на екран 1)
- Кнопка "Назад"

### Вимоги

- Навігація між 3 екранами
- Передача ID завдання як аргумент: `"details/{taskId}"`
- ViewModel для управління даними (List завдань)
- Додавання, видалення, редагування завдань
- Кнопки "Назад" та `popBackStack()`
